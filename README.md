# Internship Task 1 – Excel Sales Dashboard

## Objective
To create an interactive Excel Sales Dashboard using a Superstore-style dataset and analyze sales and profit performance across different categories, regions, and months.

## Tools Used
- Microsoft Excel
- Pivot Tables
- SUMIFS Formula
- Conditional Formatting
- GitHub

## Steps Performed
1. Collected and cleaned the sales dataset.
2. Organized the data into a structured format.
3. Created pivot-style summary tables for analysis.
4. Calculated Sales and Profit metrics using SUMIFS formulas.
5. Designed KPI cards to display key business metrics.
6. Added bar charts, pie charts, and line charts for visualization.
7. Applied conditional formatting to highlight high and low performers.
8. Created a professional dashboard for business insights.

## Results
Successfully developed an interactive sales dashboard that provides:
- Total Sales Analysis
- Total Profit Analysis
- Category-wise Performance
- Region-wise Performance
- Monthly Sales Trends
- Visual Business Insights

## Files Included
- superstore_dashboard.xlsx
- superstore_dashboard.pdf

## Conclusion
This project helped in understanding data analysis, dashboard design, data visualization, and business reporting using Microsoft Excel.
